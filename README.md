# KopifilesNAS

KopifilesNAS is a lightweight local web app that detects external media, classifies it (Pocket/Drone/Foto), and copies files into a structured folder layout with real-time progress and events.

## Overview
- Detects a media source and classifies by content and EXIF
- Copies into `DEST\YYYY\YYYY-MM-DD-City` (with optional city from EXIF/archive)
- Web UI for manual/auto source selection, formats, and archive control
- Real-time status and copy events in the UI
- Telegram notifications on completion

## Short Description
Local NAS copy manager with auto classification and real-time UI.
